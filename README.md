# 💊 FarmaLuz

Plataforma web para comparar precios de medicamentos contra 
los precios techo legales del CNFRPM/ARCSA en Ecuador.

## 🎯 Objetivo
Detectar sobreprecios ilegales en farmacias y sugerir 
alternativas genéricas más baratas al ciudadano ecuatoriano.

## 🛠️ Stack Tecnológico
| Capa | Tecnología |
|---|---|
| Backend | FastAPI (Python) |
| Base de datos | MongoDB Atlas |
| Frontend | Next.js + Tailwind CSS |
| Scraping | Playwright + Pandas |
| IA / Chatbot | Google Gemini SDK + LangChain |
| Infraestructura | AWS EC2 + Ubuntu |
| CI/CD | GitHub Actions |
| Testing | Pytest + RTL + Cypress + Jest |
